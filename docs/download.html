Windows

Tabliato est disponible pour Windows et fonctionne sous Windows 7. Il n'a jamais été testé sur d'autres Windows mais il ne devrait pas y avoir de problème. En cas de problème contactez moi. Tabliato n'est pas développé par une entreprise mais par un particulier humain qui ne mord pas.

  1. Téléchargez et décompressez l'archive au format zip
  2. Vous devez installer préalablement Lilypond en éxécutant le fichier lilypond-2.16.2-1.mingw.exe qui se trouve dans l'archive
  3. Lancez Tabliato en executant le fichier tabliato.bat

Tabliato ne fonctionne pas avec Lilypond 2.18.2. Si vous avez déjà installé cette version changez en ou contactez moi.

Version : 1.0
Architecture : 64 bits
Taille : 44 Mo

Mac

Aucune version pour Mac n'est présentement disponible. Aucun portage vers Mac n'est prévu.
