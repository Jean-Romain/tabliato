Linux

Tabliato est disponible pour toutes les distributions GNU/Linux dérivées de Debian comme Ubuntu, Kubuntu ou Linux Mint.
Téléchargez et exécutez le paquet .deb. Tout est automatique.

Version : 1.0.1
Taille : 2.3 Mo

Windows

Tabliato est disponible pour Windows et fonctionne sous Windows 7 et 10. Il n'a jamais été testé sur d'autres Windows mais il ne devrait pas y avoir de problème. En cas de problème contactez moi.

  1. Vous devez installer préalablement Lilypond
  2. Téléchargez tabliato.zip en suivant le lien ci-desosus
  3. Décompressez l'archive zip
  4. Lancez Tabliato en executant le fichier tabliato.exe

Version : 1.0.1
Taille : 33 Mo

Télécharger Tabliato pour Windows.

Mac

Aucune version pour Mac n'est présentement disponible. Aucun portage vers Mac n'est prévu. Si vous avez les compétences pour le faire, vous pouvez télécharger le code source et compiler le logiciel pour Mac. Tabliato est normalement compatible pour tous les systèmes d'exploitation. Tenez moi au courant.
