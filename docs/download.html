Linux

Tabliato est disponible pour toutes les distributions GNU/Linux dérivées de Debian comme Ubuntu, Kubuntu ou Linux Mint.
Téléchargez et exécutez le paquet .deb. Tout est automatique.

Windows

Tabliato est disponible pour Windows et fonctionne sous Windows 7. Il n'a jamais été testé sur d'autres Windows mais il ne devrait pas y avoir de problème. En cas de problème contactez moi.

  1. Téléchargez et décompressez l'archive au format zip
  2. Vous devez installer préalablement Lilypond en éxécutant le fichier lilypond-2.16.2-1.mingw.exe qui se trouve dans l'archive
  3. Lancez Tabliato en executant le fichier tabliato.bat

Version : 1.0
Architecture : 64 bits
Taille : 44 Mo
